🎓 Programação Java - Faculdade
Disciplina: Linguagem de Programação | Profª Fernanda

Este repositório é dedicado ao armazenamento e organização de todo o material prático desenvolvido durante as aulas de Java da graduação em Ciência da Computação.
📖 Metodologia das Aulas

O aprendizado é estruturado em três etapas principais:

    Conceito: Base teórica e sintaxe.

    Exercícios: Práticas de fixação.

    Desafio: Aplicações lógicas mais complexas.

🛠️ Tecnologias e Ferramentas

    Linguagem: Java

    IDE: IntelliJ IDEA

    Versionamento: Git

📂 Organização

Os códigos estão separados por pacotes (packages) seguindo a cronologia da faculdade. Para executar qualquer exercício, navegue até a pasta src/ e localize a aula ou o desafio desejado.
